# Prueba Data Engineer SoccerSolver

Prototipo end to end que responde a la pregunta de negocio: ¿estamos pagando de mas o de menos a un jugador comparado con el mercado?

El sistema obtiene y limpia datos de sueldos, los cruza con las valoraciones de mercado que aporta SoccerSolver, calcula un rango de sueldo esperado por similitud y lo muestra en una interfaz pensada para un director deportivo sin perfil tecnico.
